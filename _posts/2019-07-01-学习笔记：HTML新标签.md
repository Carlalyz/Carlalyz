## Hi，HTML5

HTML5是Web中核心语言HTML的规范，我们想要熟悉掌握必须要了解并熟悉如何去运用html的语义标签

| 标签       | 描述                     |
| ---------- | ------------------------ |
| <!DOCTYPE> | 定义文档类型。           |
| <article>  | 定义文章。               |
| <aside>    | 定义页面内容之外的内容。 |
| <body>     | 定义文档的主体。         |

更多标签请看[http://www.w3school.com.cn/tags/index.asp](http://www.w3school.com.cn/tags/index.asp)
