当一个网页用于展示自己写的文章的时候我们常常会看到一篇占据一整个页面的情况 ![](/Carlalyz/assets/images/web/19.png)

这样子我们的网页就会显得很杂乱，进行浏览的访客就会看不清我们的具体文章分类。其实要解决这种问题很简单~

我们只用在我们所写的文章中加入这样的一段文章摘要置入代码就可以完美解决这个问题了↓

```markdown
excerpt_separator: "<!--more-->"
<!--more-->
```

这一行代码我们可以将它放在md文档的title下

```markdown
excerpt_separator: "<!--more-->"
```

这一行用于大家文章想要结束句子的后面

```markdown
<!--more-->
```
